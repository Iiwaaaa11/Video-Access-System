# Video Access Management System

A Laravel-based video access management system with admin and customer roles.

## Features

### Admin Features:
- ✅ CRUD Video Management
- ✅ Approve/Reject Video Access Requests
- ✅ Set Time-limited Access for Customers
- ✅ Manage User Accounts
- ✅ Modern Bootstrap UI

### Customer Features:
- ✅ Browse Video Catalog
- ✅ Request Access to Videos
- ✅ Watch Approved Videos
- ✅ Request Access Renewal

## Tech Stack

- **Backend:** Laravel 10+
- **Frontend:** Bootstrap 5, Blade Templates
- **Database:** MySQL
- **Authentication:** Laravel Auth
